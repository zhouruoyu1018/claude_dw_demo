# 字段设计表模板

按此模板逐行填充，**禁止删减列**。简单维度字段不进入此表。

| # | 中文名 | 角色 | 语义单元 | 词根 | tags | 字段名 | 类型 | COMMENT | 来源 |
|---|--------|------|---------|------|------|--------|------|---------|------|
| 1 | {字段中文名} | {dimension/metric/boolean} | {语义1,语义2,...} | {root1,root2,...} | {TAG1,TAG2,...} | `{final_field_name}` | {DECIMAL(38,10)/STRING} | {业务含义+单位} | {并行词根查询/单次补查/复合词根查询} |

## 列说明

| 列名 | 必填 | 说明 |
|------|------|------|
| # | 是 | 序号 |
| 中文名 | 是 | 字段中文业务名称 |
| 角色 | 是 | `dimension` / `metric` / `boolean` |
| 语义单元 | 是 | 经复合词根优先处理后的语义单元，逗号分隔 |
| 词根 | 是 | 选中的 english_abbr，逗号分隔，必须来自 lookup_table |
| tags | 是 | 对应的 tag，逗号分隔，必须来自词根查询结果 |
| 字段名 | 是 | assemble_field_names 输出的最终字段名 |
| 类型 | 是 | Hive 数据类型 |
| COMMENT | 是 | DDL 中的 COMMENT 内容 |
| 来源 | 是 | 词根查询方式：`并行词根查询` / `单次补查` / `复合词根查询` |
